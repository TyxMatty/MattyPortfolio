The PRIINT Project

(Projeto de Integração Internacional Tecnológica)

The PRIINT Project (Projeto de Integração Internacional Tecnológica) is an initiative I developed alongside my partner, in collaboration with the State University of Ponta Grossa (UEPG). Our core mission is to integrate all kinds of people into the learning process, with a strong focus on accessibility for people with disabilities.

Designed for educational institutions, the software utilizes AI to provide personalized learning experiences, adapting to each student's unique needs and preferences. Built with Python and TypeScript, and integrating LLMs like Gemini and ChatGPT, PRIINT aims to aid students from all over the world in their educational journey.

Note: The PRIINT project is currently in development. Documentation is being updated on our GitHub repository, with official updates to be posted at ThePRIINTproject.org.br.
